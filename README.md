# Quiz App 
## Getting Started  

Follow these steps to set up and run the Quiz App on your local machine.  

---

## Clone the Repository  

```sh
git clone https://github.com/HarshPatel0523/Quiz-App
cd Quiz-App
```

## Backend Setup

 1. cd Backend
```sh
cd Backend
```
 2. node express.js
```sh
node express.js
```

## Frontend Setup

 1. cd Quiz-App
```sh
cd Quiz-App
```
 2. npm run dev
```sh
npm run dev
```
